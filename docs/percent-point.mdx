# Percentage to Point

point `pt` is a fixed unit of measurement that is independent of the font size of the element in which it is used. 

1 point = 1/72 of an inch.

`percentage` value to a `point` value conversion formula:

**point value** = (**percentage value** * **font size of parent element**) / 100

For example, if the font size of the parent element is `16px`, and you want to convert a percentage value of `12.5%` to points:

point value = (12.5 * 16) / 100 = 2pt


```css {{ title: 'CSS Style Sheet' }}
.element {
  font-size: 16px; /* font size of parent element */
}

.child {
  font-size: 12.5%; /* child element has a font size of 2pt (12.5 * 16 / 100) */
}

.grandchild {
  font-size: 50%; /* grandchild element has a font size of 0.5pt (50 * 16 / 100) */
}

```

You can also use the `calc` function in CSS to convert `percentage` values to `point` values. 

```css {{ title: 'CSS Style Sheet with calc function' }}
.element {
  font-size: 16px; /* font size of parent element */
}

.child {
  font-size: calc(12.5% * 16px / 1pt); /* child element has a font size of 2pt (12.5 * 16 / 100) */
}

.grandchild {
  font-size: calc(50% * 16px / 1pt); /* grandchild element has a font size of 0.5pt (50 * 16 / 100) */
}

```
