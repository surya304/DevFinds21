# Point to Percentage

point `pt` is a fixed unit of measurement that is independent of the font size of the element in which it is used.

1 point = 1/72 of an inch.

`point` value to `percentage` value conversion formula:

**percentage value** = (**point value** * 100) / **font size of parent element**

For example, if the font size of the parent element is `16px`, and you want to convert a `point` value of `32pt` to a `percentage`:

percentage value = (32 * 100) / 16 = 200%


```css {{ title: 'CSS Style Sheet' }}
.element {
  font-size: 16px; /* font size of parent element */
}

.child {
  font-size: 32pt; /* child element has a font size of 200% (32 * 100 / 16) */
}

.grandchild {
  font-size: 8pt; /* grandchild element has a font size of 50% (8 * 100 / 16) */
}

```

You can also use the `calc` function in CSS to convert `point` values to `percentage` values. 

```css {{ title: 'CSS Style Sheet with calc function' }}
.element {
  font-size: 16px; /* font size of parent element */
}

.child {
  font-size: calc(32pt * 100% / 16px); /* child element has a font size of 200% (32 * 100 / 16) */
}

.grandchild {
  font-size: calc(8pt * 100% / 16px); /* grandchild element has a font size of 50% (8 * 100 / 16) */
}

```
